# 📝 | Installation

* [Node.js](https://nodejs.org/dist/v20.11.0/node-v20.11.0-x64.msi)

* Required installing module
```bash
npm i proxy-agent
npm i ws
```

* Install requirements

* Put your proxy in `proxy.txt`

* Clone/Download the repo

* Starting
```bash
node server.js or
node .
```
